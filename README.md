# Uber_Ride_Analysis_using_Power_BI

### **📊 One-Line Summary**

A Power BI dashboard project analyzing 150K Uber rides to uncover insights on revenue, customer behavior, cancellations, and operational performance.


### **📖 Overview**

This project focuses on analyzing Uber ride data to understand business performance, customer behavior, and operational challenges.
Using Power BI, I built an interactive dashboard that provides insights into bookings, revenue, cancellations, locations, and time-based trends.


### **❗ Problem Statement**

Uber faces multiple business challenges such as:

High ride cancellations and incomplete bookings

Low customer retention

Uneven demand across locations

Peak hour overload

Underperformance of premium vehicle categories


This project aims to identify these issues and provide data-driven insights for better decision-making.


### **📂 Dataset**

Total Records: 150,000 bookings

Key Fields:

Booking Status (Completed, Cancelled, Incomplete)

Vehicle Type

Pickup & Drop Location

Distance

Payment Method

Customer & Driver Ratings

Date & Time


### **🛠️ Tools and Technologies**

Power BI – Dashboard creation & visualization

Power Query – Data cleaning & transformation

DAX (Data Analysis Expressions) – KPIs & calculations

Excel / CSV – Data source


### **⚙️ Methods**

Data Cleaning (handling null values, formatting)

Data Transformation using Power Query

Data Modeling (relationships, measures)

KPI creation using DAX

Visual storytelling using charts, slicers, and filters

Anomaly detection for trend analysis


### **🔍 Key Insights**

62% rides completed, while 38% lost bookings

Auto & Bike contribute ~47% of total revenue

Evening (4 PM–8 PM) is peak booking time

Weekdays contribute 71% of bookings

Customer retention is very low (mostly first-time users)

Top pickup location: Khandsa

Top drop location: Ashram

UPI & Cash are the most preferred payment methods


### **📊 Dashboard / Output**

The dashboard includes multiple pages:

Overview (KPIs & performance summary)

Vehicle Analysis

Revenue Insights

Customer Behavior

Location Analysis

Time & Weekly Trends

Cancellation Analysis

It provides interactive filters for:

Vehicle type

Time period

Location


### **▶️ How to Run this Project**

Download the Uber_PowerBI_Dashboard.pbix file from this repository

Open it using Power BI Desktop

Refresh the dataset (if required)

Explore the dashboard using filters and slicers


### **📈 Result & Conclusion**

This project highlights major areas of improvement for Uber:

Reduce cancellations by improving driver allocation

Improve customer retention through loyalty programs

Optimize operations during peak hours

Focus more on high-performing vehicle types

Improve address accuracy and customer experience


Overall, this dashboard helps in making data-driven business decisions.
